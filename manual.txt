first you choose an option

Option 1 YouTube:
If you chose YouTube you can enter a searchterm for the application to find the most recent uploaded videos that match the search term.
Next you choose the amount of results you would like to get.


Option 2 ICT-Jobs:
You can enter a job title or company to look for and this will return the most recent posted job offers that match the search.
Next you choose the amount of results you would like to get.


Option 3 HTLV:
You can enter a player name, team name, event name or article title.
Next you choose the amount of results you would like to get.
The application returns the results for the search, this can contain players, teams, events, articles that match the search term.


Every time you want to scrape you have to run the code, which is not user friendly but you should not have to run a scraper over and over I guess.

How to use written in steps:
Step 1: Choose type of scraping
The scraper asks for the type of scraping you want to do (YT, ICT-jobs, HLTV).

Step 2: Choose in what language the browser being scraped should be.
The scraper asks for the language for the browser.

Step 3: Enter search term to scrape results for on website.
The scraper asks for the search term to look for on the selected website.

Step 4: Choose number of results to scrape.
Only for YouTube and ICT-Jobs. On HLTV everything will be scraped.
The scraper asks for the number of results you would like to return.

Step 5: The scraper opens browser and gets results.
The browser opens and goes to the selected website with the entered search term and scrapes the data you requested.
 
You can also see the files created by the scraper with the results in the specified path
